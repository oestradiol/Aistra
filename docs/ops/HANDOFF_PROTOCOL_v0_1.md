# Handoff protocol v0.1

## Operator procedure
1. Do not paraphrase packet items.
2. Use the local app when possible.
3. Export the JSON file immediately after completion.
4. Keep packet version, scorer version, and answer-key version attached.
5. Do not manually rescore without documenting why.

## Participant procedure
1. Answer from the stimulus, not current feeling.
2. Fill the 9 scored fields.
3. Do not infer source/certainty/contour beyond explicit support in baseline-explicit mode.
4. Export and return the file.

## File naming
Use the default export naming convention from the app.
If manually renaming, preserve:
- route
- date-time
- package version

## Return bundle
Minimum return artifact:
- exported results JSON

Preferred return artifact:
- results JSON
- human-readable report markdown
- notes on any known misclicks or interruptions
