# Pilot red-team findings v0.1

Major issues found in the original pilot layer:
1. Same underlying targets appeared in both structured and baseline conditions for the same participant.
2. Condition order was fixed, creating avoidable order bias.
3. Baseline wording was too close to direct canonical paraphrase.
4. Advanced notation (`?`, `Ø`) appeared too early in the main packet.
5. Practice materials were implied but not concretely packaged.

Action taken:
- drafted red-teamed v0.2 protocol, baseline memo, review template, and pilot packet
- reduced scored main packet to simpler items
- separated practice from scored items
- removed within-participant target overlap across conditions
- made baseline wording more natural and less slot-mechanical

Recommendation:
Do not run v0.1 as written.
Use v0.2 as the recommended first pilot layer.
